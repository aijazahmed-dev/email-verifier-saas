# Email Verifier SaaS

A professional Django-based Email Verifier designed to help businesses and developers ensure their email lists are clean and deliverable. 

## Features
- Syntax validation
- MX record lookup
- Disposable email detection
- Role-based email detection
- SMTP verification (deliverability check)
- Catch-all and greylisting handling
- Bulk CSV upload & clean list download
- REST API for integration
- Web dashboard with templates and Tailwind CSS

## Tech Stack
- Backend: Django, Django REST Framework
- Database: PostgreSQL (Dockerized)
- Frontend: Django Templates + Tailwind CSS
- Deployment: Docker + docker-compose
- Optional: Email verification API integration

## Purpose
- MVP for product monetization (micro-SaaS)
- Showcase project for recruiters
- Can be sold or deployed for recurring revenue
